# Prerequisites
```
yum install -y epel-release && yum makecache
yum install -y binutils coreutils util-linux socat ethtool iptables ebtables ipvsadm ipset psmisc bash-completion conntrack-tools iproute nfs-utils 
yum install -y wget ca-certificates
# install .so for iproute2
yum install -y zlib pcre libselinux libmnl libdb libcap libattr iptables glibc elfutils-libelf
```
